# CromaScan 🎨

CromaScan은 이미지에서 대표 색상을 추출하고  
어울리는 색상 조합을 추천해주는 **Android 애플리케이션**입니다.

카메라 촬영 또는 갤러리 이미지를 기반으로 색상 팔레트를 생성하고,  
추천 색상 조합을 시각적으로 확인할 수 있습니다.

---

## 📱 주요 기능

- 📷 **카메라 촬영 / 갤러리 이미지 선택**
- 🎨 **K-means 알고리즘 기반 대표 색상 추출**
- 🧩 **선택한 색상에 어울리는 추천 색상 조합 제공**
- 🔁 **추천 색상 클릭 시 해당 색상을 기준으로 재추천 기능**
- 💾 **색상 조합 히스토리 저장 및 조회**
- 🖼️ **결과 화면 이미지 공유 기능**

## ▶️ 실행 방법

1. Android Studio에서 프로젝트 열기
2. 에뮬레이터 또는 **실제 Android 기기 연결**
3. `Run ▶` 버튼 클릭
4. 앱 실행 후 이미지 선택 또는 카메라 촬영

※ 카메라 기능은 **실제 기기에서 정상 동작**합니다.
